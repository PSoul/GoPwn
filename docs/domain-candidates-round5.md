## 行业命名 Pattern 总结

| 模式 | 示例 | 特征 |
|------|------|------|
| xxGPT | PentestGPT, RamiGPT, HackTricksGPT | 直白AI标签，但 GPT 是 OpenAI 商标 |
| xxsploit | Metasploit, autosploit | 经典安全圈后缀，exploit 缩写 |
| xxscan | fscan, masscan, tangscan | 扫描类直接 |
| 品牌造词 | Nuclei, Nebula, Garak, Caldera | 科幻/科学，无明确安全语义但辨识度高 |
| xx+sec | keensec, razorsec | 安全后缀 |
| xx+pwn | darkpwn, redpwn | 黑客文化 |
| xx+hunt | redhunt, vulnhunt | 狩猎/猎人 |
